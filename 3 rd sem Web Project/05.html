5.VLC


  • Originally called VideoLAN Client, VLC media player is the main software of the VideoLAN solution.
  • VideoLAN was originally designed for network streaming but VideoLAN’s main software, VLC media player has evolved to become a full-featured cross-platform media player.
  • VideoLAN is a complete software solution for video streaming and playback, developed by students of the Ecole Centraleand developers from all over the world, under the GNU General Public License.
  • VideoLAN is designed to stream MPEG videos on high bandwidthnetworks.
  • works on many platforms: Linux, Windows, Mac OS X, BeOS, *BSD, Solaris, Familiar Linux, Yopy/Linupy and QNX. It can play:
  • VCDs, and Audio CDs
  • acquisition or encoding cards (on GNU/Linux and Windows only).
  • MPEG-2 and MPEG-4 / DivX files from a hard disk, a CD-ROM drive, ...
