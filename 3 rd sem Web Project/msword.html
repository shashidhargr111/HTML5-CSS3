MICROSOFT WORD


  • Microsoft Word is an example of a program called a “word processor".
  • Microsoft Word is a very powerful word processor.
  • Word processors are used to create and print text documents in much the same way that you would use a typewriter.
    • The key benefit to using a word processor is that you can make changes easily, including correcting spelling; adding, deleting, formatting, and relocating text; and inserting images.
  • Once you create a document, you can effortlessly print it (as many copies as you want!), save it for later modifications, or send it to a friend via e-mail.
